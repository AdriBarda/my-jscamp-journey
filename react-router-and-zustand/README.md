## Overview

This folder contains the main React version of the project. It uses Vite for fast
local development (HMR) and for generating a production build.

## Scope

- React components and client-side routing
- Vite dev server and build pipeline
- Front-end only (no backend or database at the moment)

## Run

1. Install dependencies: `pnpm install`
2. Start dev server: `pnpm dev`

Optional:

- Build for production: `pnpm build`
- Preview build locally: `pnpm preview`
