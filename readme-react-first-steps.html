React: First Steps

A bridge stage between the static prototype and the Vite app, using React directly in the browser to understand the fundamentals before moving to the standard toolchain.

React Babel Standalone JavaScript No Bundler

Overview

This folder exists to demonstrate how React can be introduced into an existing project in a very direct way. It keeps the setup lightweight so the focus stays on JSX, components, state, and rendering behavior.

Scope

  • React rendered in the browser via CDN-style imports
  • Simple components and local state with useState
  • An educational step, not the production setup recommended for real apps

Run

Open react-first-steps/react.html in the browser Inspect the JSX and Babel-in-browser setup
