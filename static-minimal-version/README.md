## Overview
This folder contains a minimal static implementation of the project UI. It is meant for
layout validation, visual review, and content structure exploration without any backend
or framework dependencies.

## Scope
- Pure HTML, CSS, and vanilla JavaScript
- No API calls, database, or server-side rendering
- No framework, package manager, or build pipeline

## Run
Open `index.html` directly in the browser. Navigation between pages uses standard links.
